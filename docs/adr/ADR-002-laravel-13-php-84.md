# ADR-002 — Laravel 13 com PHP 8.4

```yaml
status: aceita
date: 2026-06-14
```

## Decisão

- Laravel 13;
- PHP 8.4.

## Motivo

Laravel 13 exige PHP 8.3 ou superior.

PHP 8.4 oferece baseline estável e compatível sem depender imediatamente da versão mais nova do runtime.

## Consequência

Toda dependência precisa suportar PHP 8.4 e Laravel 13.
