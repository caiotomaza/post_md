# Escopo do repositório web

## Responsabilidades

Este repositório implementa:

- aplicação Laravel;
- interface web;
- API HTTP;
- autenticação;
- pastas;
- notas Markdown;
- tags;
- imagens e vídeos;
- busca;
- histórico;
- importação e exportação;
- compartilhamento;
- administração;
- jobs;
- storage;
- observabilidade;
- operação Docker.

## Fora do escopo

- aplicação Android;
- Kotlin;
- Room;
- WorkManager;
- publicação de APK;
- UI mobile nativa.

O mobile consumirá a API deste repositório quando o contrato estiver estável.

## Primeira entrega

A primeira entrega deve priorizar:

1. ambiente;
2. banco;
3. layout web;
4. pastas;
5. notas;
6. editor;
7. tags;
8. testes.

Usuários e compartilhamento entram depois do núcleo.
