## Laravel Boost

Depois que o Laravel estiver instalado, o projeto pode avaliar:

```bash
composer require laravel/boost --dev
php artisan boost:install
```

A instalação deve ser revisada antes do commit.

- testes antes de concluir;
- não descartar mudanças não commitadas;
- atualizar status e documentação.
